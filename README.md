# Matrix Style Portfolio Website

A React + Vite website concept with a Matrix-inspired visual style and EmailJS contact integration.

This project demonstrates frontend layout work, interactive UI ideas, and basic email form integration for a portfolio or personal website.

## Features

- React + Vite project structure
- Matrix-inspired visual design
- Contact form integration with EmailJS
- Welcome email flow for users
- Responsive website concept

## Tech Stack

- React
- Vite
- JavaScript / TypeScript
- HTML
- CSS
- EmailJS

## Why I Built This

I built this project to practice modern frontend development, animated visual concepts, and real-world contact form behavior.
